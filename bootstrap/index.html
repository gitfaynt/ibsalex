input Inline semantisch Formularelement, verwendbar für diverse Typen wie Eingabefelder, Schaltflüchen, Auswahlfelder usw.
ins Block semantisch Eingefügter Inhalt (Änderungsmarkierung)

label Inline semantisch Beschriftungstext eines Formularfeldes

link Inline semantisch Logische Beziehungen der aktuellen Webseite zu anderen Webseiten
map Inline semantisch Bereich für verweis-sensitive Grafiken
meta semantisch Meta-Angabe für nicht im Inhalt sichtbare Dokumentinformationen oder für Anweisungen an den Webserver
noscript Block semantisch Bereich für Inhalte, die angezeigt werden, wenn JavaScript im Browser deaktiviert oder nicht verfügbar ist
object Inline semantisch Eingebettetes Objekt, geeignet für spezielle Multimedia-Objekte, Java-Applets usw.

option Inline semantisch Auswahlmüglichkeit in einer Auswahlliste
p Block semantisch Paragraph, Absatz
param Inline semantisch Parameter eines Objekts bei object-Element

q Inline semantisch Würtliche Rede in einem Text

sup Inline semantisch Hochgestellter Text

td semantisch Tabellenzelle
textarea Inline semantisch Mehrzeiliger Eingabebereich in einem Formular
tfoot Inline semantisch Fuübereich in einer Tabelle, z.B. für summarische Daten
th Block semantisch Tabellenkopfzelle
thead Block semantisch Kopfbereich einer Tabelle
title Inline semantisch Titel eines HTML-Dokuments
tr Block semantisch Tabellenzeile
tt semantisch üFernschreiberü-Text
ul Block semantisch Unsortierte Liste (Aufzühlungsliste)
